# DeepSeek-V3 ассистент

📖 **Подробная пошаговая инструкция — [ИНСТРУКЦИЯ.md](ИНСТРУКЦИЯ.md)** (установка, запуск,
все команды, как выжать максимум).

Личный ИИ-ассистент в терминале — **как Claude Code, но бесплатный**. Работает на
**DeepSeek-V3 (671B)** через бесплатные **GitHub Models** (мозг в облаке — твой ПК не
считает модель). Умеет реально работать с компьютером: читать и править файлы, искать по
коду, выполнять команды. Есть **PLAN MODE** (как у меня): сначала план, потом действия.

## Как это устроено

```
ты (терминал) ─►  assistant.py  ─►  GitHub Models (DeepSeek-V3) в облаке
                      │
                      ├─ инструменты: read_file, list_dir, search,
                      │               write_file, edit_file, run_shell
                      └─ PLAN MODE: только чтение + план, без изменений
```

- **Мозг** — DeepSeek-V3 в облаке GitHub Models. Бесплатно по твоему GitHub-токену.
- **Надёжность** — если у модели кончился дневной лимит, ассистент сам переключается на
  следующую (DeepSeek-V3 → gpt-4o → gpt-4o-mini), не встаёт.
- **Безопасность** — перед изменением файла или командой спрашивает подтверждение
  (`y/N/a`). В PLAN MODE изменения запрещены вовсе.

## Установка (один раз)

```bash
cd "/mnt/c/Users/Mick Thomson/deepseek-assistant"
pip install -r requirements.txt          # нужен только requests
```

Токен уже прописан в `.env` (твой GitHub-токен, в git не попадает). Заменить — см.
`.env.example`.

Удобный запуск одной командой (добавь алиас в `~/.bashrc`):

```bash
echo "alias assist='python3 \"/mnt/c/Users/Mick Thomson/deepseek-assistant/assistant.py\"'" >> ~/.bashrc
source ~/.bashrc
```

## Как пользоваться

Зайди в папку, с которой хочешь работать, и запусти:

```bash
cd /путь/к/твоему/проекту
assist                 # (или: python3 /путь/.../assistant.py)
```

Дальше просто пиши задачу, как мне. Ассистент сам почитает нужные файлы, внесёт правки,
выполнит команды (спросив подтверждение).

### Команды

| Команда | Что делает |
|---|---|
| `/plan` | включить **PLAN MODE** — только чтение и план, ничего не меняет |
| `/run`  | выключить PLAN MODE — разрешить правки и команды |
| `/yolo` | не спрашивать подтверждение на каждое действие (вкл/выкл) |
| `/reset`| очистить историю диалога |
| `/help` | список команд |
| `/exit` | выход |

### Пример

```
ты › изучи проект и предложи, что улучшить
[ассистент читает файлы и отвечает]

ты › /plan
ты › добавь логирование в main.py
[ассистент выдаёт план, ничего не трогая]

ты › /run
ты › теперь сделай по плану
[ассистент правит файлы, спрашивая подтверждение]
```

## Честно про возможности

DeepSeek-V3 — сильнейшая бесплатная модель, для большинства задач (код, тексты, анализ,
автоматизация) — отличный помощник. На самых сложных многошаговых задачах топовый Claude
ещё немного впереди, и у бесплатного тира GitHub Models есть суточные лимиты на модель
(поэтому встроено авто-переключение). ПК твой не нагружается — считает облако.
